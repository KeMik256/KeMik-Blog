[极少量实体]

Level 0 是一个辽阔的非欧几里得空间，外貌类似于零售店的密室仓库。整个Level 0的所有房间都具有相同特征：单调的黄色墙纸、老旧潮湿的地毯、分散的电线插座，以及不同位的荧光灯。除了这些普遍的特点之外，Level 0内部没有两个房间是完全相同的。

Level 0 中的荧光灯会以恒定的频率嗡嗡作响，这种嗡嗡声比普通荧光灯更响亮、更突兀。我们对荧光灯的检测进行了多次，但是它的来源迄今没有任何定论。其次，使地毯轻微湿透的液体不能被辨别：但这不是水，也不能安全饮用，其成分类似于胸膜液。

Level 0 中的线性空间被极大的扭曲了；你可以选择沿着直线走，然后回到起点，最后进入与之前经过的房间完全不同的房间。由于这种现象，以及每个房间之间的视觉相似性，因此在Level 0内导航被证明是极其困难的。诸如GPS定位器和罗盘等设备在Level 0中无法发挥作用，无线电通信也会失真，而且往往不可靠。

在Level 0 中流浪者常常会出现以下幻觉：

  1. 观察到门、楼梯、窗户、电梯的出现；
  2. 观察到其他人类的出现；
  3. 荧光灯声音突然震耳欲聋，随后又归于寂静；
  4. 突发的即视感；
  5. 余光观察到虫在墙纸上爬动；
  6. 听到虫鸣声。

Level 0 中最严酷的威胁是可用资源的严重缺乏。由于浸透地毯的液体被认为对人类食用不安全，因此大多数人可能会由于长期脱水、饥饿和筋疲力竭而在找到出口之前死亡。

避开那些闪烁着昏暗红光的房间，那是红室的入口，任何进入红室的流浪者均被认为再无出路。

Level 0被认为是完全不存在生命的。该楼层没有到流浪者在探索期间与其他流浪者或实体接触的记录。如果你遇到了一个像人类的生物，远离它，它不是人类。
